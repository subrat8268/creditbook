# Archive

This folder contains **tool-specific or legacy AI setup files** that are intentionally **not part of the active OpenCode + OMO workflow** for KredBook.

Why this exists:
- Reduce beginner confusion.
- Keep one authoritative AI operating system.
- Preserve historical guidance without letting it compete with the active system.

Active AI entrypoints live at:
- `AGENTS.md`
- `SYSTEM_CONTEXT.md`
- `_agents/`
