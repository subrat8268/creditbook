# frontend-design

Create distinctive, production-grade frontend interfaces. No more generic AI slop.

*   Pick a bold aesthetic direction (brutalist, editorial, luxury, retro-futuristic...)
*   Choose distinctive typography (never Arial or Inter)
*   Constrained color palettes (1 primary + 1 accent + neutrals)
*   Micro-interactions on every hover/focus
*   Skeleton loading states, never spinners

**Install:** [github.com/anthropics/claude-code/tree/main/plugins/frontend-design](http://github.com/anthropics/claude-code/tree/main/plugins/frontend-design)

**Use when:** "Build me a landing page" / "Design a dashboard" / "Create a marketing site"
