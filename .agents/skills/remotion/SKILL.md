# remotion

Create videos programmatically with React. Reels, ads, carousels -- all code-driven.

*   30+ rule files for animations, transitions, timing, captions
*   spring() for physics-based motion
*   Export: MP4, GIF, PNG sequence, ProRes
*   Media rules for images, audio, fonts, 3D, Lottie, charts

**Install:** [remotion.dev/docs/ai/skills](http://remotion.dev/docs/ai/skills)

**Use when:** "Create an Instagram ad" / "Build a video intro" / "Make an animated explainer"
