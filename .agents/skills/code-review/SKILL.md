# code-review

Multi-language code reviewer covering React 19, Rust, TypeScript, Python, Go, and more.

*   4-phase review: context gathering, architecture check, line-by-line, summary
*   Severity labels: [blocking], [important], [nit], [suggestion], [learning], [praise]
*   Language-specific guides loaded on demand
*   Golden rule: ask questions, don't command. Always explain WHY.

**Install:** [github.com/awesome-skills/code-review-skill](http://github.com/awesome-skills/code-review-skill)

**Use when:** "Review this PR" / "Check this code for security issues" / "Review my React components"
